# 🗳️ Seçim Pusulası | Election Compass

A political questionnaire app that helps users discover which Turkish political party aligns with their views — inspired by [DR.dk's election compass](https://www.dr.dk/nyheder/politik/folketingsvalg/kandidattest).

**[Try it live →](https://onbuyuka.github.io/tr-poli-questionnaire/)**

## How It Works

1. Answer 19 policy-focused questions across 7 categories
2. Rate each statement from *Strongly Disagree* to *Strongly Agree*
3. Get ranked results showing your match percentage with each party

Questions are about **policy topics**, not about specific politicians or party loyalty.

## Categories

| Category | Topics |
|---|---|
| 🏦 Economy | State role, minimum wage, progressive taxation |
| 🕌 Secularism | Religion in governance, religious education, Diyanet |
| ⚖️ Democracy | Judicial independence, press freedom, parliamentary system |
| 🌍 Foreign Policy | EU membership, NATO, refugee policy |
| 👥 Social Issues | Women's rights, LGBTI+ rights, minority rights |
| 🌱 Environment | Renewable energy, environmental protection |
| 🕊️ Kurdish Issue | Democratic resolution, language rights |

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Hosted on GitHub Pages

## Development

```bash
npm install
npm run dev       # Start dev server at localhost:3000
npm run build     # Production build
npm run deploy    # Build & deploy to GitHub Pages
```

## Disclaimer

This application is independent and has no affiliation with any political party. Party positions are approximate assessments based on publicly stated programs and may not be exact. This tool is for informational purposes only.
